# Dotfiles

My personal configuration files for Omarchy (Arch Linux with Hyprland)

## Contents

- `.config/hypr/` - Hyprland window manager configuration
- `.config/waybar/` - Waybar status bar configuration
- `.config/alacritty/` - Alacritty terminal configuration
- `.config/ghostty/` - Ghostty terminal configuration
- `.config/nvim/` - Neovim configuration
- `.config/omarchy/` - Omarchy-specific configurations
- `.config/walker/` - Walker application launcher
- `.config/btop/` - btop system monitor
- `.config/fastfetch/` - fastfetch system info
- `.config/git/` - Git configuration
- `.config/starship.toml` - Starship prompt configuration

## Installation

To use these dotfiles:

```bash
git clone https://github.com/YOUR_USERNAME/dotfiles.git ~/dotfiles
cd ~/dotfiles
# Then create symlinks or copy files to ~/.config/
```

## System

- OS: Arch Linux
- WM: Hyprland
- Bar: Waybar
- Terminal: Alacritty / Ghostty
- Shell: Bash/Zsh with Starship prompt
- Editor: Neovim
